利用WaitGroup并发访问DB

适用场景

在之前写的项目中，当需要查询多个数据库/数据表时，不论他们之间没有先后关系的要求，都是统一一个接着一个去访问db，这种方式对于先后次序无关的查询或者操作来说效率太低，我们可以考虑并发的去访问数据库来进行优化
